Profile Card

This project is a visually engaging and fully responsive user profile card built from scratch using only HTML and CSS.

Project Overview

A comprehensive profile card component designed to showcase user information in an elegant and interactive way.

Key Features

  • Responsive Design: The profile card is built to be fully responsive, using a flexible layout that adapts seamlessly to various screen sizes.
  • Interactive Elements: The main action buttons ("My Works", "Contact me") provide clear visual feedback to the user with smooth transitions and a change in appearance on hover.
  • Flexbox for Layout: The entire component is structured using CSS Flexbox, ensuring robust and precise alignment of the profile image, text data, buttons, and analytics section.
